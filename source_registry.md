## 三、各机构类型详细说明

### S1 国资委（国资监管机构）

**职能定位**: 代表本级政府履行出资人职责，对授权监管范围内的国有企业实施监管。

**关键判断**:
- 省级: 31个省级行政区均设有独立国资委
- 地市级: 约70%设有独立国资委，其余由财政局代行或设国资中心
- 县区级: 约20%设有独立国资委/国资办，其余多由财政局国资股管理
- 直辖市的区级: 通常有区国资委

**常见栏目名称** (来自实地采样):
| 栏目 | 对应信息 | 采样来源 |
|------|----------|----------|
| 政务公开 / 信息公开 | 综合入口 | 通用 |
| 监管企业 / 出资企业 / 企业名录 | Part1 名单 | 北京:国企之窗→监管企业/双管企业 |
| 国资数据 / 运行数据 / 经济运行 | Part2 运行 | 国务院国资委:国资数据（央企/国企/地方/行业） |
| 监管动态 / 国企改革 / 资本运营 | Part3 重组等 | 通用 |
| 政策文件 / 通知公告 | 综合政策 | 通用 |
| 规划计划 / 考核分配 | 战略与考核 | 部分省市 |

**URL模式** (省级):
- `gzw.{province}.gov.cn` — 最常见（如 gzw.beijing.gov.cn, gzw.sc.gov.cn）
- `gzw.{province}.gov.cn/xxfb/` — 政务公开入口
- `gzw.{province}.gov.cn/yggq/jgqy/` — 监管企业（北京模式）

**注意事项**:
- 监管企业名单 ≠ 全部国有企业名单（只含授权监管范围内的一级企业）
- 部分地方存在"双管企业"（经营性事业单位转企业）
- 省级国资委网站通常列有本省市州国资委链接

### S2 财政厅/财政局

**职能定位**: 编制国有资本经营预算；管理行政事业性国有资产；对未纳入国资委监管的国有企业（文化、金融等）行使出资人职责。

**关键判断**:
- 国有资本经营预算的编制主体是财政部门，不是国资委
- 行政事业性国有资产管理的归口部门是财政
- 金融类国有企业通常由财政部门（而非国资委）监管
- "四本预算"之一的国有资本经营预算必须在财政部门查找

**常见栏目名称**:
| 栏目 | 对应信息 |
|------|----------|
| 预算公开 / 预决算公开 | Part4 预算 |
| 国有资本经营预算 | Part4 专项 |
| 资产管理 / 行政事业资产 | Part4 资产报告 |
| 财政数据 / 财政收支 | Part2 补充（汇总财政口径） |

**URL模式**:
- `czj.{province}.gov.cn` 或 `czt.{province}.gov.cn`（厅/局差异）
- 预决算公开专栏通常在一级导航

### S3 人民政府门户

**职能定位**: 综合信息发布平台，转发各部门文件；部分重大政策（如企业重组批复）以政府名义发文。

**关键判断**:
- 政府门户上的内容多为**转载/转发**，需追溯原始发布机构
- 重组批复、国企改革方案等以"X政函"编号的文件，原始发布在政府门户
- 新闻通稿中的数据需验证是否为正式发布

**常见栏目名称**:
| 栏目 | 对应信息 |
|------|----------|
| 政府信息公开 / 政务公开 | 综合入口 |
| 政策文件 / 规范性文件 | 重组批复等 |
| 机构设置 / 组织机构 | ⚠️ 高风险误判：不含业务数据 |
| 新闻发布 / 政务动态 | ⚠️ 新闻稿非原始数据来源 |

### S4 人大常委会门户

**职能定位**: 审议政府专项报告（含国有资产管理情况报告）；监督预算执行。

**关键判断**:
- 2018年起，全国人大建立国有资产管理情况报告制度，地方人大逐步跟进
- 人大审议的是**政府提交的报告**，人大网站上的是审议版本
- 审议意见/审议发言 与 政府正式公开稿 可能存在差异
- 人大预算工委有独立的预算审查监督职能

**常见栏目名称** (来自 npc.gov.cn 实地采样):
| 栏目 | 对应信息 |
|------|----------|
| 权威发布 → 报告 | Part4 国资报告（中央层面） |
| 监督 → 工作监督 / 审议发言 | Part4 审议过程 |
| 预算工作委员会 | 预算监督专项 |
| 财政经济委员会 | 经济监督综合 |

**URL模式** (地方人大):
- `www.{province}rd.gov.cn` — 省级人大（如 bjrd.gov.cn, shrd.gov.cn）

### S5 预决算公开统一平台

**职能定位**: 集中公开各级各部门预算/决算信息。

**关键判断**:
- 中央: http://www.mof.gov.cn/yjsgkpt/ (财政部预决算公开平台)
- 地方: 各省有独立预决算公开平台或在财政厅网站设专栏
- 国有资本经营预算**草案/执行/决算**三个阶段都应在此查找
- 预算目录页（目录索引）和预算正文（PDF/附表）是两回事 ⚠️

### S6 国有企业官网

**职能定位**: 企业自主披露信息。

**关键判断**:
- 发债企业: 必须在中国债券信息网等平台披露财务报告
- 上市企业(含子公司上市): 在交易所网站披露
- 非公开企业: 信息披露不规范，需结合产权交易所等渠道
- 重组/合并公告通常第一时间在企业官网发布

### S7 审计厅/审计局

**职能定位**: 依法审计国有资产、国有资本经营预算执行情况。

**常见栏目名称**:
| 栏目 | 对应信息 |
|------|----------|
| 审计结果公告 | 可包含国企运营审计发现 |
| 预算执行审计 | Part4 相关 |
| 经济责任审计 | 国企负责人任期审计 |

### S8 国资事务中心/国有资产经营公司

**职能定位**: 在未设独立国资委的地区，承担国有资产管理实际职能的事业单位或国有平台公司。

**关键判断**:
- 通常无独立官网，信息在财政局或政府门户子栏目
- 可能是实际掌握企业名单的唯一机构
- 行政层级低，信息公开规范性差

## 四、检索优先级树

### 4.1 Part1 — 监管国有企业名单

```
Step 1: 判断该地区是否设有独立国资委
  ├─ 有国资委 → 直接查 S1 国资委 > "监管企业"/"出资企业" 栏目
  │    └─ 补充查 S3 政府门户 > 机构设置（仅确认国资委存在，不取数据）
  ├─ 无国资委 → 判断替代机构
  │    ├─ 财政局国资股 → 查 S2 财政局 > 国有资产管理 栏目
  │    ├─ 国资中心/国资办 → 查 S8 > 如有官网则查；否则查S3政府门户
  │    └─ 政府授权经营公司 → 查 S6 该平台公司官网
  └─ 不确定 → 先查 S3 政府门户 > 机构设置 确定出资人机构类型
```

### 4.2 Part2 — 国有企业月度/年度运行

```
Step 1: 国资委口径数据
  S1 国资委 > "国资数据"/"经济运行"/"统计数据" 栏目
Step 2: 财政口径补充
  S2 财政厅/局 > 国有资本经营预算执行数据
Step 3: 企业自报
  S6 国企官网 > 年报/经营数据
Step 4: 审计验证
  S7 审计厅 > 国有资本审计报告
```

### 4.3 Part3 — 重组/基金/主责主业

```
Step 1: 国资委是首选
  S1 国资委 > "监管动态"/"资本运营"/"国企改革" 栏目
Step 2: 政府批复文件
  S3 政府门户 > 政策文件（X政函/X政发编号）
Step 3: 企业公告
  S6 国企官网 > 重组公告/主业调整公告
Step 4: 产权交易平台
  C类第三方 > 各地产权交易所网站
```

### 4.4 Part4 — 国有资本经营预算 / 国有资产管理情况报告

```
Step 1: 预算类 → 财政为主
  S2 财政厅/局 > 国有资本经营预算草案/执行/决算
  S5 预决算公开平台 > 同上
Step 2: 资产报告类 → 人大为主
  S4 人大门户 > 国有资产管理情况报告（审议版本）
  S3 政府门户 > 政府提交版本（如有单独发布）
Step 3: 国资委补充
  S1 国资委 > 综合运营数据（企业经营性国有资产部分）
Step 4: 审计验证
  S7 审计厅 > 国有资本经营预算执行审计结果
```

### 6.3 Part1 采样验证详情

#### REG-002 北京市国资委 — Part1 详细采样

**监管企业页**:
- 路径: 首页 > 国企之窗 > 监管企业
- URL: `gzw.beijing.gov.cn/yggq/jgqy/`
- 分页: 共3页（每页约10家企业）
- 模式: LIST-A（每企业一个详情页链接）
- 详情页内容: 企业简介（叙事文本，约200-500字），含企业历史、主营业务、发展方向
- 详情页字段: 页面顶部有"时间"（如2021年06月17日）和"来源"（市国资委）
- 采样企业: 首钢集团/北京公交/北京基础设施投资/北京地铁运营/北控集团/北京能源集团/首创集团/北京国资公司/北京自来水/北京银行/华夏银行/北京金控/北京农商行/北京电控/北汽集团/北京建工/北京城建/首旅集团/首都实业/京城机电/北京一轻/北京时尚/北京化工/同仁堂/北京金隅/首开集团/北京北辰/首发集团/祥龙资产/北京排水

**双管企业页**:
- 路径: 首页 > 国企之窗 > 双管企业
- URL: `gzw.beijing.gov.cn/yggq/sgqy/`
- 分页: 共3页
- 内容: 中央企业驻京单位/分公司（如国网北京电力/中石化北京分公司/中国铁塔北京分公司/北京烟草/厦门航空北方总部等）
- ⚠️ 注意: 双管企业并非北京市出资企业，而是在京央企的属地监管

**关键发现**:
- 北京银行/华夏银行/金融控股等金融类企业也在监管名单中 → 北京是金融类纳入国资委监管的特例
- URL中日期(201812)≠内容实际更新日期(2021-06) → 不能用URL日期作为有效日期
- 双管企业是北京特有分类概念

- **标题模式**: "{year}年1-{N}月全国国有及国有控股企业经济运行情况" (极度一致)
- **来源标注**: "文章来源：财政部网站"

- **历史覆盖**: 可追溯至少到2023年（3页文章列表）
- **格式**: RUN-A，4个标准指标分节:
  1. 营业总收入: 绝对值(亿元) + 同比%

  3. 应交税费: 绝对值(亿元) + 同比%
  4. 资产负债率: %(MM月末) + 同比变动(个百分点)
