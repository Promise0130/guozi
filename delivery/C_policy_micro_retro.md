# C_policy_micro_retro — 政策简报模块微复盘

> 日期: 2026-04-02
> 模块: Module C — 政策动态简报

## 暴露的问题

### C3-01 政策主题误检
- **表现**: 品牌建设意见(POL-003)被纳入"主责主业管理"，但其核心是品牌管理而非主业聚焦
- **根因**: 主题词库过宽，"主责主业"的命中条件未排除品牌/ESG/社会责任等外围主题
- **分类**: **Skill 缺陷**
- **建议**: 补主题词排除列表

### C3-02 排除词不足
- **表现**: 学习教育、政绩观、读书班等大量新闻被初始抓取返回，占据网页内容主体
- **根因**: 排除词列表缺乏"学习教育""政绩观""读书班""述职""团拜会"等高频噪声词
- **分类**: **Skill 缺陷**
- **建议**: 补政策采集排除词库

### C3-03 文种识别不稳
- **表现**: "工作要点"与"新闻通稿"边界模糊——如黑龙江35项制度建设的报道，是"新闻通稿"还是"工作要点"？
- **根因**: 缺乏文种自动分级规则，目前依赖手工判断
- **分类**: **Skill 缺陷**
- **建议**: 补文种分级判定规则（基于标题关键词+发文机关+正文结构特征）

### C3-04 约束性表述提取不稳定
- **表现**: "应当""不得""必须"等约束词能准确识别，但"鼓励""原则上""可以"等弱约束词的分级不清
- **根因**: 缺乏约束性表述的强度分级标准
- **分类**: **Skill 缺陷**
- **建议**: 建立约束性表述四级分级（强制/应当/鼓励/可选）

### C3-05 地区间表达差异大
- **表现**: 同一类政策在不同省份的命名差异极大
  - 中央: "实施办法" → 黑龙江: "实施办法" (一致)
  - 中央: "指导意见" → 福建: "五大行动" (完全不同)
  - 中央: "管理办法" → 宁波: "三提行动" (完全不同)
- **根因**: 地方政策命名不遵循中央文件体例
- **分类**: **Skill 缺陷** + 数据本身特征
- **建议**: 补地方政策文种映射规则

### C3-06 司库体系主题采集薄弱
- **表现**: 仅采集到1条直接相关政策(POL-007)，且全文未公开
- **根因**: (a) "司库"表述在政策文件中出现频率低，实际用"财务管理体系""资金集中管理"等同义词;
            (b) 2022年的世界一流财务管理体系指导意见是核心基础文件但不在2025年窗口内
- **分类**: **Skill 缺陷**（同义词不足）+ **数据本身缺失**
- **建议**: 补司库体系同义词映射

### C3-07 证据强度分布不均
- **表现**: 12条政策中，S1级4条(33%)、S2级1条(8%)、S3级5条(42%)、S4级2条(17%)
- **根因**: 政府网站信息公开层级不同；省级政策全文公开率极低
- **分类**: **基础设施限制** + 数据本身特征
- **建议**: 证据强度低的条目标注"待人工补充"

### C3-08 时间窗口边界问题
- **表现**: POL-006(廉洁从业规定)和POL-007(财务数智化)发文日期为2026年3月，但修订/起草过程横跨2025年
- **根因**: "2025年以来"的时间边界如何处理跨年度文件不明确
- **分类**: **Skill 缺陷**
- **建议**: 补时间窗口判定规则（以发布日期为准 vs 以起草/生效日期为准）
